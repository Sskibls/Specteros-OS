FEATURES

BUILT FOR PRIVACY

Advanced security features engineered into every layer

🎭

Persona Shards

Isolated environments for different identities. Each shard has independent storage, network routes, and security policies.

Learn more →
⚡

Network Kill Switch

Emergency network disconnect at hardware level. One keystroke blocks all traffic to protect your identity instantly.

Learn more →
🔐

Full Disk Encryption

LUKS2 encryption by default. Your data remains protected even if physical security is compromised.

Learn more →
📋

Audit Logging

Tamper-evident security event logging with cryptographic verification. Know exactly what happened and when.

Learn more →
🧅

Tor Integration

Built-in Tor support for anonymous browsing. Route any application through Tor with transparent proxying.

Learn more →
🛡️

Secure Boot Chain

UEFI Secure Boot with custom key enrollment. Complete boot chain verification from firmware to userspace.

Learn more →
🔄

A/B System Updates

Seamless updates with automatic rollback. Never brick your system - always bootable, always recoverable.

Learn more →
🔒

DNS over HTTPS

Encrypted DNS queries by default. Prevent DNS snooping, manipulation, and surveillance by ISPs.
