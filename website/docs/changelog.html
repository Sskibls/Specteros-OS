Changelog

v0.1.0 Latest
March 5, 2024

Initial Public Release

The first public release of SpecterOS, featuring core privacy and security functionality.

  • Added Persona Shards system with 4 isolated environments (Work, Anon, Burner, Lab)
  • Added Network Kill Switch for emergency network disconnect
  • Added LUKS2 full disk encryption support
  • Added Tamper-evident audit logging framework
  • Added Tor integration with transparent proxying
  • Added DNS over HTTPS support
  • Added A/B system updates with rollback support
  • Added XFCE desktop environment with custom theme
  • Added Privacy-focused web browser (Firefox ESR hardened)
  • Added Terminal dashboard (ghostkernel-tui)
  • Added Interactive shell (ghostkernel-shell)
  • Added Web-based control interface
  • Fixed Mobile website layout issues
  • Fixed Network daemon crash on startup
  • Changed Improved installer user interface
  • Changed Updated documentation with installation guide
v0.0.3-alpha Stable
February 25, 2024

Beta Testing Release

Final beta release before public launch.

  • Added Emergency modes (Panic, Mask, Travel)
  • Added Guardian daemon for OPSEC alerts
  • Fixed Shard isolation boundary issues
  • Fixed Memory leak in audit daemon
  • Changed Improved boot performance
v0.0.2-alpha
February 15, 2024

Alpha Release

Second alpha release with core functionality.

  • Added Policy daemon with capability-based permissions
  • Added Airlock daemon for secure file transfers
  • Added Metadata sanitizer for images and documents
  • Fixed Network routing between shards
v0.0.1-alpha
January 30, 2024

First Alpha

Initial alpha release for internal testing.

  • Added Base Debian 12 system
  • Added Core security daemons
  • Added XFCE desktop environment
  • Added Basic installer
